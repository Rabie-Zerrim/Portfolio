## Common Issues & Solutions

### "Site not indexed after 2 weeks"
- Check robots.txt allows crawling
- Verify sitemap is accessible
- Request indexing again
- Check for manual penalties

### "Ranking dropped"
- Check for broken links
- Ensure site is mobile-friendly
- Add more content
- Get more backlinks

### "Not ranking for 'Rabie'"
- Too competitive? Try "Rabie Zerrim" first
- Add more mentions of your name
- Get backlinks with anchor text "Rabie"
- Create content about yourself
